Crave

Crave is an app where you can search for the best restaurant for a specific dish. Users can sign in to vote up their favorite restaurant, or post a new restaurant that hasn't been added.


This app was built from concept to completion on a team of three in four days. We utilized the Yelp API to pull in restaurant information and built our own API using Express.js with a MongoDB database. Our front-end and user authentication was created using Angular.js.

My personal contributions include:

  • Creating the front-end user authentication with Angular.js
  • Accessing the Yelp API
  • Styling with CSS

Listage

Listage is an app for venues to list live music events. Venues can sign in post information about upcoming shows for concert goers to view.


This app was built from concept to completion on a team of three in four days. We built a RESTful API using Express.js with a MongoDB database. Our front-end and user authentication was created using Angular.js.

My personal contributions include:

  • Creating a RESTful API using Express.js
  • Utilizing a Mongo database to store venue information and show listings
  • Creating the front-end user authentication with Angular.js

  • See Listage in Action

Giftr

Giftr is an app that allows users to store and view lists of gift ideas for friends. After creating an account, a user can add friends to a list, then create, edit or remove gift ideas.


This app was created from concept to completion on a team of four in four days. We utilized Firebase to store user information and provide user authentication.

My personal contributions include:

  • Creating an Express.js server
  • Utilizing Node.js Event Emitters to dynamically add and remove friends and gift items to list
  • Utilizing jQuery on the front-end to manipulate DOM elements

  • See Giftr in Action
